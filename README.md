## 정리

주소 API는 대략 크게 2가지 정도
1) 다음 API
2) 공공 API

### Daum API vs. Korea Public Data(public API)

Duam API
- 가장 많이 사용
- Vue, React에서 사용할 수 있도록 github에 올려줌(나도 사용)
- kakao(daum)에 문서가 잘 됭 있음
- pc / mobile 웹 지원
- 가장 최신의 데이터 반영(도로명, 영문, 지번주소까지 모두 확인 가능)

Korea Public Data
- 인증키를 발급받아야 사용 가능
